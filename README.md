# Virtual Escape Room using Data Structures

A Java console-based puzzle adventure game that demonstrates core Data Structures and Algorithms through interactive gameplay.

## Features
- 5 Puzzle Rooms
- Score and Lifeline System
- Mini-puzzles and riddles
- Time-based challenges

## Data Structures Used
- Stack
- Array / ArrayList
- HashMap
- Graph (BFS)
- Recursion
- Queue

## Rooms
1. Stack Puzzle
2. Hangman Game (HashMap)
3. Maze Escape (Graph BFS)
4. Tower of Hanoi (Recursion)
5. Cipher Adventure (Array logic)

## Technologies
- Java
- Data Structures & Algorithms
- Console-based interface

## How to Run

```bash
javac *.java
java GameMain
